# orc
Orchestration of agents: agentic AI

## Agents
The user can create an agent with a persona and specializations, along with tools they can use. 
The user has a view of all the agents, along with the task they are working on, if any.
The agent has a memory of the tasks they have completed, along with the actions they attempted, both successful and unsuccessful.
The user can erase the memory of an agent.

## Tasks
An agent can be given a task. The user decides if the task must terminate after some time, or if they can run until the user interrupts.
The user can ask the agent to report on the task status at any moment.
The user can stop a task at any moment.
The user has a view of all running tasks.
